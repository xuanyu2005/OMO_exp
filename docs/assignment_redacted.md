# 作业要求（脱敏版）

建议实验方向：

- 对比 `gpt5.4mini` 和 `claude sonnet 4.6` 组合是否会比单一模型更好。

- 维度二：带 OMO 插件但所有 agent 都配置为同一模型，是否比不同模型组合更差。
- 分析带来提升或没有提升的原因。
